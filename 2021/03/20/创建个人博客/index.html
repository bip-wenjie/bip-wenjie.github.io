创建个人博客

March 20, 2021 667

1. Node.js下载

  • 地址 ： nodejs.org

  • 安装，全部next!

  • 验证：win + R ==> cmd 进入命令行

    node -v
    npm -v
    显示版本号则成功

  • npm install -g cnpm --registry=https://registry.npm.taobao.org
    验证：
    cnpm -v

2. 安装hexo

cnpm install -g hexo-cli
验证
hexo -v

  • 创建博客目录:在hexo目录下，创建blog目录

  • 初始化
    alt
    hexo init

  • 本地测试
    hexo -s
    复制 http://localhost:4000 去浏览器打开查看
    至此最简单的个人博客已经做好

3. 部署博客至github
